# Loan-Default

📊 Credit Risk Insights: Loan Default Analysis Dashboard

1. Project Title / Headline
💳 Credit Risk Insights: Loan Default Analysis Dashboard
An interactive Power BI dashboard designed to analyse borrower behaviour, identify default risk patterns, and support data-driven lending decisions through intuitive visualisations.

2. Short Description / Purpose

The Credit Risk Insights Dashboard is a dynamic Power BI report built to explore loan default trends, borrower profiles, and financial risk indicators. It helps financial institutions and analysts better understand the factors influencing loan repayment and default probability.

3. Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Core platform for building interactive reports and visualisations
📂 Power Query – Used for data cleaning, transformation, and shaping
🧠 DAX (Data Analysis Expressions) – Implemented for calculated measures, KPIs, and dynamic logic
📝 Data Modelling – Relationships between borrower, loan, and repayment datasets for accurate analysis
📁 File Format – .pbix for development and report sharing


4. Data Source

Source: Financial loan dataset (www.kaggle.com)

The dataset includes detailed information on:

Borrower demographics (age, income, employment status, etc.)
Loan attributes (loan amount, term, interest rate, purpose)
Repayment status (default, fully paid, ongoing)
Credit history and risk indicators

The structured data enables analysis of default trends across multiple dimensions such as income level, loan size, and borrower profile.

5. Features / Highlights
   
• Business Problem
Loan defaults pose a significant financial risk to banks and lending institutions. Without clear visibility into borrower behaviour and risk factors, it becomes difficult to:
Identify high-risk applicants
Optimise loan approval strategies
Reduce financial losses due to defaults

Key questions include:
Which borrower segments are most likely to default?
How do income and loan amount impact repayment behaviour?
What patterns exist across different loan categories?

• Goal of the Dashboard

To deliver an interactive analytics tool that:

Identifies key drivers of loan defaults
Enables segmentation of borrowers by risk level
Supports better credit decision-making
Provides insights for risk mitigation strategies
• Walkthrough of Key Visuals

🔹 Key KPIs (Top Section)

Total Loans Issued
Total Defaults
Default Rate (%)
Average Loan Amount
Average Interest Rate

🔹 Default Distribution (Bar / Pie Chart)
Shows proportion of defaulted vs. non-defaulted loans.

🔹 Loan Amount vs Default (Scatter / Column Chart)
Analyses how loan size impacts default likelihood.

🔹 Income vs Default Rate (Line / Bar Chart)
Highlights the correlation between borrower income levels and default behaviour.

🔹 Loan Purpose Analysis (Bar Chart)
Breakdown of defaults across loan purposes (e.g., personal, education, business).

🔹 Credit Risk Segmentation (Clustered Chart)
Segments borrowers into low, medium, and high-risk categories.

🔹 Filters / Slicers Panel
Interactive filters for:
Loan status
Income range
Loan purpose
Term duration

• Business Impact & Insights
📉 Risk Reduction: Identify high-risk borrowers before loan approval
🎯 Better Decision-Making: Enable data-driven credit policies
📊 Customer Segmentation: Understand borrower profiles and behaviour
💡 Strategic Planning: Improve loan products and pricing strategies
💼 Operational Efficiency: Reduce manual risk assessment efforts
